# NeuroGuide

A research companion for machine learning practitioners — discover papers, audit datasets, and generate production-ready ML code.

## Features

- **Research Paper Discovery** — Find and organize relevant ML papers with smart search
- **Dataset Auditing** — Analyze and validate datasets for quality and bias
- **ML Code Generation** — Generate production-ready code from research concepts
- **Project Management** — Track research projects with health scores and status
- **AI Chat Interface** — Interactive assistant for research guidance

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (Database, Auth, Edge Functions)
- **Build:** Vite
- **Styling:** Tailwind CSS with custom design system

## Getting Started

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project
cd neuroguide

# Install dependencies
npm install

# Start development server
npm run dev
